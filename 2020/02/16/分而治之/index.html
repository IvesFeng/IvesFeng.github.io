分而治之

发表于 2020-02-16 Valine：

分而治之(Divide and conquer)定义:

分而治之的工作原理是将问题递归分解为两个或多个相同或相关类型的子问题，直到这些子问题变得足够简单以至于可以直接解决。然后将子问题的解决方案组合起来，以解决原始问题。

分治法步骤:

分治法有三个步骤:

​ 分解(divide) 原问题为若干子问题, 这些子问题是原问题的规模较小的实例.

​ 解决(conquer) 递归地求解子问题, 如果子问题的规模足够小, 则停止递归, 直接求解.

​ 合并 把子问题的解合并成原问题的解

分治法示例:

1. 归并排序

分解: 分解待排序的n个元素的序列成各具n/2个元素的两个子序列

解决: 使用归并排序递归地排序两个子序列

合并: 合并两个已排序的子序列以产生已排序的答案

归并排序的关键步骤在于第三步, 如何合并两个已排序的序列.

假设有两个序列a = [4, 5, 10], b = [1, 3, 9], 合并后放入c

第一步, 分别取两个序列中的第一个元素2和1, 然后把较小的放入c, 得到a = [4, 5, 10], b = [3, 9], c = [1]

重复第一步取出3和4比较, 可以得到a = [4, 5, 10], b = [9], c = [1, 3]

继续重复第一步取出4和9比较, 得到a = [5, 10], b = [9], c = [1, 3, 4]

最后可以得到c = [1, 3, 4, 5, 9, 10]
